## 1. 微信公众号（认证服务号）

| 序号 | 项目 | 老板填写 | 备注 |

## 2. 腾讯云 COS（对象存储）

| 序号 | 项目 | 老板填写 | 备注 |
|------|------|----------|------|
| 2.1 | **SecretId** | | 主账号或子账号 API 密钥 |
| 2.2 | **SecretKey** | | 仅创建时完整可见，请安全保存 |
| 2.3 | **地域 Region** | 例：`ap-guangzhou` | 与桶所在地域一致 |
| 2.4 | **存储桶 Bucket 全名** | 例：`xxx-1250000000` | 控制台 COS 桶列表可复制 |
| 2.5 | 该密钥对目标桶权限是否包含 **上传/读取** | ☐ 已确认 | 按最小权限由运维配置 |

**H5 网页直传视频时**：需在 COS 控制台为该桶配置 **CORS**（允许公司 H5 域名、`PUT`、所需 Header）。可写「由运维/开发联调时配置」。
