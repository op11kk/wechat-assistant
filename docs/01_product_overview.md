# EgoCapture 产品概述

## 产品定位
面向具身智能 / 人形机器人训练的 **egocentric 多模态数据采集 APP**。  
对标参考：Stray Scanner，但更偏训练数据生产与交付。

## 核心目标
采集、保存、上传可训练的 **RGB / Depth / IMU / Pose** 数据，将 session 结构化保存、管理和上传到后台。

## 一句话定义
EgoCapture 是一个 iPhone 端数据采集器，用于稳定采集第一视角的 RGB、深度、IMU 和位姿数据，并将 session 结构化保存、管理和上传到后台，服务于机器人训练数据生产。

## 产品目标（G1~G6）
| 编号 | 目标 |
|------|------|
| G1 | 在支持设备上稳定采集 egocentric 训练数据 |
| G2 | 支持 Full Capture 与 Lite Capture 双模式 |
| G3 | 保证数据具备统一时间基准，便于下游对齐 |
| G4 | 支持单次至少 10 分钟稳定录制，目标 10-30 分钟 |
| G5 | 支持本地保存、查看、重试上传和后台入库 |
| G6 | 避免竞品常见问题：发热、漂移、变速、文件损坏、自动丢失 |

## 非目标（V1 不做）
- Android 采集端
- 端侧模型训练
- 自动标注
- 数据售卖平台前台
- 复杂视频编辑
